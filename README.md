# Filtex
A trivial tool for moving files with a specific extension.

## Installation
```
go get github.com/danielwelch/filtex
go install
```

## Usage
```filtex /path/to/old_directory/ /path/to/new_Directory/ .extension```

## Example
```filtex ~/Documents/ ~/Dropbox/ .txt```
